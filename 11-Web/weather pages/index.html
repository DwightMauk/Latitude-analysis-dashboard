Summary

While using Open Weather API and Citipy library I pulled 500 random cities to create a dataset to analyze how weather patterns may change from city to city based on latitude closeness to the Equator.

After collecting the data, I used matplotlib to analyze the data set. I set up four varibles which were; Max Temp, Cloudiness, Humidity, and Wind Speed. This page shows the source data and the visualizations based on the four dependent variables.
